## Useful Packages and Libraries

Extend your Go applications with these popular packages:

### Web Frameworks

<CardGroup cols={2}>
  <Card title="Gin" icon="wine-glass" href="https://gin-gonic.com/">
    Fast HTTP web framework with a martini-like API.
  </Card>
  
  <Card title="Echo" icon="volume-high" href="https://echo.labstack.com/">
    High performance, minimalist Go web framework.
  </Card>
  
  <Card title="Fiber" icon="network-wired" href="https://gofiber.io/">
    Express-inspired web framework built on top of Fasthttp.
  </Card>
  
  <Card title="Chi" icon="code-branch" href="https://go-chi.io/">
    Lightweight, idiomatic and composable router for building HTTP services.
  </Card>
</CardGroup>

### Database Libraries

<CardGroup cols={2}>
  <Card title="GORM" icon="database" href="https://gorm.io/">
    Full-featured ORM library for Go with support for multiple databases.
  </Card>
  
  <Card title="sqlx" icon="table" href="https://github.com/jmoiron/sqlx">
    Extensions to Go's standard database/sql library.
  </Card>
</CardGroup>

### Testing & Development Tools

<CardGroup cols={2}>
  <Card title="Testify" icon="vial" href="https://github.com/stretchr/testify">
    Toolkit with common assertions and mocks for testing.
  </Card>
  
  <Card title="Air" icon="wind" href="https://github.com/cosmtrek/air">
    Live reload for Go apps during development.
  </Card>
</CardGroup>

### Logging & Monitoring

<CardGroup cols={2}>
  <Card title="Zap" icon="bolt" href="https://github.com/uber-go/zap">
    Blazing fast, structured, leveled logging in Go.
  </Card>
  
  <Card title="Zerolog" icon="terminal" href="https://github.com/rs/zerolog">
    Zero allocation JSON logger.
  </Card>
</CardGroup>
